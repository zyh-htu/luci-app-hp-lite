# luci-app-hp-lite
hp-lite客户端适用于openwrt的ipk安装包

# 安装说明
到 release 页面下载最新版的hp-lite（注：请根据你的路由器架构下载对应版本）

将文件上传到你的路由器上，进行安装

opkg install luci-app-hp-lite*.ipk
安装完毕，你可以在 /usr/bin 目录下找到对应的二进制文件。
